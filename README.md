This script:

- reads temperature
- stores temperature on a SD CARD
- trigger a relay (heater) if the temperature is within a range

 * SD card attached to SPI bus as follows:
 ** MOSI - pin 11
 ** MISO - pin 12
 ** CLK - pin 13
 ** CS - pin 4 (for MKRZero SD: SDCARD_SS_PIN)
